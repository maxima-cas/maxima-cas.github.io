StageTools - Documentation (CenterStage)

CenterStage Home Page:

The CenterStage module lets you create Geomview objects using mathematical formulas. It can generate parametric surfaces and curves, polyhedra, vectors, and combinations of these (e.g., tubes around curves, offset surfaces and curves, etc.). These can be combined into hierarchical, compound objects. Objects can include parameters that are dependent on sliders, check-boxes, type-in areas, and other input devices for their values. StageManager can be used to control these values, so that, for example, you can create a movie that shows how a surface deforms in response to changes in its parameters.
