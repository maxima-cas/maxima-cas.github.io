StageTools - Documentation (Download)

Downloading StageTools:

StageTools can be obtained from the Math Department web server at Union College. The distribution is via a gzipped tar file, which you should be able to unpack on most unix systems. Since StageTools is a suite of Geomview modules, and since Geomview only runs under unix, StageTools can only be used on a unix system. While the StageTools code could run on a Macintosh or IBM/PC, you would not be able to see the output without some replacement for Geomview.

  • StageTools-2006-08-17.tgz (1.8 M)

Once you have downloaded and unpacked the file, follow the installation instructions. These also are available in the StageTools/doc directory of the archive you have just downloaded.
