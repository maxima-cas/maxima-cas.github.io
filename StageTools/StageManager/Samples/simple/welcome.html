Rotating Cube:

This examples shows how to load an object into Geomview and set its initial view (via the Transform commands). Finally, it shows how to perform a simple action in the movie: a rotation about the z axis.

The cube file used in this script is a standard one that comes with Geomview.

cube.sm

   Load cube
   Transform {YZ -$pi/3}
   Transform {XY -$pi/6}

   Sequence {XY 2*$pi} 30
