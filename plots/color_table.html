Plot Colors

The following table shows the 139 named colors that can be used in Maxima plots, and their corresponding values in hexadecimal red-green-blue code. These are the same colors defined in HTML, CSS and SVG. You can of course use any color code #xxxxxx with 6 hexadecimal digits x, but these 139 colors can be referred by name. Instead of "gray", you can also write "grey"; "fuchsia" can be used as a synonym for "magenta", and "aqua" is accepted as synonym for "cyan".
