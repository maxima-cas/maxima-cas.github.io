HTML Colors

The following table shows the 139 names of colors accepted in HTML, and their corresponding values in hexadecimal red-green-blue code. You can of course use any color code #xxxxxx with 6 hexadecimal digits x, but these 139 colors can be referred by name. Instead of "gray", you can also write "grey". "fuchsia" can be used as a synonym for "magenta", and "aqua" is accepted as synonym for "cyan". I have ordered the colors by value, from the ones that look brighter to me, to the ones that look darker.
