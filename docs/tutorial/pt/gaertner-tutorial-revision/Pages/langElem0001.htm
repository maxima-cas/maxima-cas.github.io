Iniciando

Maxima é um sistema em linha de comando. Você informa um comando, obtém uma resposta e pode inserir o comando seguinte.

Um comando é uma expressão que pode ocupar mais de uma linha e é encerrado com um ponto e vírgula. e.g.

(%i1) 14*(7 + 3);

Você obtém essa resposta:

(%o1)  140

Note que nada irá acontecer quando você esquecer de colocar o ponto e vírgula. Maxima irá simplesmente esperar por entradas adicionais.

Você pode inserir uma expressão que contém variáveis:

(%i2) (x + 2)*(x - 3);

Maxima responde a essa expressão da forma simplificada:

(%o2)  (x - 3) (x + 2)

Note que você tem que digitar um asterisco para denotar a multiplicação. Maxima prefere uma forma de escrever  que não utiliza um operador de multiplicação: Como na notação tradicional de matemática, a multiplicação está implícita.

Talvez você possa preferir ver o produto expandido. Isso é alguma coisa que que você tem que perguntar. 

expand(%);
	   2
(%o5)	  x  - x - 6

Note a interpretação de expoentes:

x^3^4;
	      81
(%o2)	     x
(x^3)^4;
	      12
(%o4)	     x

Vemos que x^3^4 é avaliado como x^(3^4), não como (x^3)^4.
