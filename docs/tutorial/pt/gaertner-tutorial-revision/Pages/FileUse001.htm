Uso de Arquivos

Existem muitos mecanismos para armazenar dados em arquivos e recuperá-los.

  1. save escreve dados em um arquivo como expressões Lisp. load recupera os dados escritos por save. Execute ? save e ? load para mais informações.

  2. stringout escreve dados em um arquivo como expressões Maxima. load recupera dados escritos por stringout. Execute ? stringout para mais informação.

  3. writefile cria um console de log (um então chamado arquivo "dribble") que grava as interações entre o usuário e Maxima. Esse arquivo não pode chamado novamente no Maxima. Execute ? writefile para mais informações.

  4. As funções read_matrix, read_list, read_nested_list, etc., podem ler arquivos que contêm números ou símbolos, arranjados em linhas e separados com caracteres de espaços, vírgulas, ou pipe (barra vertical). write_data escreve dados no mesmo formato.

    Essas funções estão no pacote numericalio. load (numericalio) torna essas funções disponíveis para o usuário.

    Os comentários no arquivo share/contrib/numericalio/numericalio.lisp fornecem mais detalhes sobre essas  funções.
