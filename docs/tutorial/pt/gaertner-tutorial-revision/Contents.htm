Esse tutorial abrange o uso do Maxima. Além de material próprio, esse tutorial usa - de forma modificada - material da homepage http://www.ma.utexas.edu/users/wfs de William F. Schelter (veja: http://en.wikipedia.org/wiki/Bill_Schelter), o último mantenedor do Maxima.

Onde começar:

Se você não tem uma cópia do Maxima, mas quer ter uma, você pode ler informações de Download. Para realmente entender o Maxima, você terá que trabalhar com esse grande programa. Instalar o Maxima é simples, não hesite em tentar instalar!

Para uma primeira tentativa no Maxima, você pode querer tentar os exemplos em Primeiros Passos com Maxima.

O uso de um programa de álgebra computacional não é sempre fácil. O iniciante sente muitas vezes um pouco incapaz quando ele precisa de  uma função para executar uma operação especial. É necessário exercitar o uso do Maxima para tornar-se um especialista. É uma boa idéia tentar reslver exercícios dos livros didáticos.
